HTML5 Content Separator

Tag Description
<address> Used to wrap content that contains contact information for an individual or organization.
<aside> Wraps elements that are only indirectly related to the document's main content, often used for sidebars or call-out boxes.
<footer> Denotes the footer of a document, which typically contains copyright notices or similar content.
<header> Contains introductory content such as a logo or author information for a section or article.
<nav> Contains navigation links, often presented as an unstyled list in a top bar or sidebar.
<main> Holds the primary content of the document. There should only be one <main> per page, and it typically excludes headers, footers, and navigation.
<article> Wraps content that can stand alone and be distributed independently, like blog or forum posts.
<section> Used to group related content under a heading. For example, organizing departments and their employees.
